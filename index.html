Available API Endpoints

Method Path Description
GET/booksGet all books in the library
GET/books/genre/:genreGet all books of a specific genre
GET/books/genre/:genre/id/:idGet a book by genre and ID
GET/books/genre/:genre/author/:authorGet books by genre and author
GET/books/genre/:genre/title/:titleGet books by genre and title
GET/books/author/:authorGet all books by a specific author
GET/books/cupboard/:cupboardGet all books in a cupboard (1–5)
GET/books/shelf/:shelfGet all books on a shelf (a–e)
GET/books/cupboard/:cupboard/shelf/:shelfGet books at exact location e.g. cupboard 1 shelf a = 1a
GET/books/issuedGet all currently issued books
GET/books/issued/issuedBy/:userGet all books issued to a specific user
GET/books/availableGet all books available for borrowing
GET/books/:idGet a single book by its unique ID
